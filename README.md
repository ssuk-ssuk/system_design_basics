단순한 시스템 설계에서 출발하여 시스템 확장에 따른 이슈와 요구사항을 파악하자!

# 스터디 방법

매주 정해진 분량을 읽고, 스터디 진행전 학습한 내용을 정리하여 개시합니다.

발표자는 무작위로 정해지며 연속으로 발표할 수 있습니다.

학습을 할 때에는 정해진 분량을 제외하여 궁금했던 내용들을 확장하여 학습하는 것을 권장합니다.

ex) A를 활용하면 B를 해결할 수 있어요. → 사이드 이펙트는 없나? → 다른 방법은 없나? → 예시 코드를 한번 만들어볼까??
